\# NeuroTeach AI



An AI-powered teaching engine that transforms academic PDFs into an interactive learning experience.



\## Features



\- 📄 Upload academic PDFs

\- 🧠 AI-generated concept explanations

\- 🗣 Hinglish / English teaching modes

\- ⌨ Typing-style teaching flow

\- 🌙 Light / Dark mode support

\- 💾 Study session tracking

\- 📚 Progressive concept learning



\## Tech Stack



\- Streamlit

\- FLAN-T5 (Transformers)

\- PyMuPDF

\- Sentence Transformers



\## How it Works



1\. Upload an academic PDF

2\. AI filters out metadata \& headings

3\. Concepts are taught progressively

4\. Explanations are generated in teacher-style language

5\. Users can continue learning from saved progress
